Seattle Collision Heatmap



This map was a group project with four other students in GEOG 328 (Web-Based GIS). Using GeoJSON data obtained from the Seattle Department of Transportation, I contributed to developing an interactive heatmap displaying vehicle collision severity in Seattle. The original dataset had 20 years of collision data, so for storage purposes it was cut down to a 21 month period starting in March 2022 and ending in December 2023.

Development stack:
  • Mapbox API
  • Python (GeoPandas)
  • HTML
  • CSS
  • JavaScript

Dataset: SDOT Collisions All Years
